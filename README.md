# Laszlo Romsics

Software Developer and Data Scientist building across iOS, macOS, backend systems, 
and data pipelines. Final-year BSc Mathematics student at The Open University. 
Founder of [EastSideLabs](https://github.com/EastSideLabs).

Based in Bournemouth, UK.

## About

- Apple platform development: iOS and macOS with Swift
- Backend and infrastructure: self-hosted Docker stacks, Traefik, Linux VPS
- Data science and ML: statistical modelling, PyTorch, scikit-learn
- Comfortable across the full stack from database to UI
